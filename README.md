# siyuan-pseudocode

一款让你在思源笔记中优雅地书写和展示伪代码的挂件。

> 思源笔记伪代码挂件(siyuan-pseudocode)最初由[Samon Yu (samonysh)](https://github.com/samonysh/)开发，之后与本人开发的siyuan-latex-pseudocode挂件合并，重新改名为siyuan-pseudocode。感谢Samon Yu的贡献。

## 功能特征

- [x] 无网络离线使用
- [x] 设置伪代码块的显示风格
- [x] VSCode风格的编辑器
- [x] 语法高亮
- [x] 代码自动补全
- [x] 错误提示
